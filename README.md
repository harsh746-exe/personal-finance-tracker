# Personal Finance Tracker

A comprehensive Streamlit-based web application for managing personal finances, tracking investments, managing debts, and setting financial goals.

## Features

### Core Financial Management
- 📊 Track income and expenses with detailed categorization
- 💰 Set and monitor category-specific budgets
- 📈 Visualize spending patterns with interactive charts
- 📅 Monthly financial trend analysis
- 💡 Financial management tips
- 📱 Responsive web interface

### Investment Portfolio
- 📈 Track stocks, mutual funds, ETFs, and other investments
- 📊 Real-time stock price tracking using Yahoo Finance API
- 📉 Portfolio performance visualization
- 💰 Investment returns calculation
- 📊 Asset allocation analysis

### Debt Management
- 💳 Track various types of debts (credit cards, loans, etc.)
- 📅 Generate debt payment schedules
- 📊 Interest rate calculations
- 📈 Debt payoff visualization
- 💰 Monthly payment tracking

### Financial Goals
- 🎯 Set and track financial goals
- 📊 Progress visualization
- 📅 Target date tracking
- 💰 Goal categories (Emergency Fund, Retirement, etc.)
- 📈 Progress tracking with visual indicators

### Bill Management
- 📅 Track recurring bills and payments
- 🔔 Bill due date reminders
- 💰 Bill categorization
- 📊 Payment history
- 📈 Bill payment timeline

### Advanced Analytics
- 📊 Comprehensive financial reports
- 📈 Investment portfolio analysis
- 📉 Debt-to-income ratio calculation
- 💰 Net worth tracking
- 📅 Monthly financial summaries

### Data Management
- 💾 Export financial data to JSON
- 📥 Import financial data from JSON
- 🔄 Data backup and restore
- 📊 Custom report generation
- 📈 Historical data analysis

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/finance-tracker.git
cd finance-tracker
```

2. Create a virtual environment (recommended):
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. Install the required packages:
```bash
pip install -r requirements.txt
```

4. Run the application:
```bash
streamlit run finance_tracker.py
```

## Usage

1. **Dashboard**: View your overall financial status, including income, expenses, investments, and debts
2. **Transactions**: Add and track income and expenses
3. **Investments**: Manage your investment portfolio and track performance
4. **Debts**: Track and manage your debts with payment schedules
5. **Goals**: Set and monitor your financial goals
6. **Bills**: Manage recurring bills and payments
7. **Reports**: Generate comprehensive financial reports and export data
